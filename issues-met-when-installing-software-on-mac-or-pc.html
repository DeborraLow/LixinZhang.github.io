Issues met when installing software on Mac or PC

解决Ubuntu下ImportError: libblas.so.3gf

如果你import scipy时出现，类似： 

import scipy
Traceback (most recent call last):
  File "", line 1, in 
  File "/usr/lib/python2.7/dist-packages/scipy/init.py", line 78, in 
    from numpy import show_config as show_numpy_config
  File "/usr/lib/python2.7/dist-packages/numpy/init.py", line 137, in 
    import add_newdocs
  File "/usr/lib/python2.7/dist-packages/numpy/add_newdocs.py", line 9, in 
    from numpy.lib import add_newdoc
  File "/usr/lib/python2.7/dist-packages/numpy/lib/init.py", line 13, in 
    from polynomial import 
  File "/usr/lib/python2.7/dist-packages/numpy/lib/polynomial.py", line 17, in 
    from numpy.linalg import eigvals, lstsq
  File "/usr/lib/python2.7/dist-packages/numpy/linalg/init.py", line 48, in 
    from linalg import 
  File "/usr/lib/python2.7/dist-packages/numpy/linalg/linalg.py", line 23, in 
    from numpy.linalg import lapack_lite
ImportError: liblapack.so.3gf: cannot open shared object file: No such file or directory

那么尝试安装libatlas-base-dev，至少我越到的问题解决了。

sudo apt-get install libatlas-base-dev
