    Add ssh keys to github

    创建SSH key

    ssh-keygen
生成的SSH key文件保存在中～/.ssh/id_rsa.pub

    添加SSH key到github

    接着拷贝.ssh/id_rsa.pub文件内的所以内容
打开github帐号管理中的添加SSH key界面的步骤如下：
1. 登录github
2. 点击右上方的Accounting settings图标
3. 选择 SSH key
4. 点击 Add SSH key
