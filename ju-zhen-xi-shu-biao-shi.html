矩阵稀疏表示

最近用theano中的DBN做东西，但是由于训练集比较大，直接将feature按01展开，放不下内存。于是需要用到稀疏矩阵表示Feature。

import theano
import numpy as np
import scipy.sparse as sp
from theano import sparse

a = sp.csc_matrix(np.asarray([[0, 0, -1], [0, -2, 1], [3, 0, 0]], dtype='float32'))

对一个稀疏矩阵增加行，可以使用hstack和vstack

from scipy.sparse import vstack, hstack
a = sp.csc_matrix(np.asarray([[0, 0, -1], [0, -2, 1], [3, 0, 0]], dtype='float32'))
b = sp.csc_matrix(np.asarray([[0, 0, -1], [0, -2, 1], [3, 0, 0]], dtype='float32'))
c = vstack((a, b)) # 纵向相加
d = hstack((a, b)) # 水平相加
